Raspberry Pi Homescreen

The Raspberry Pi Homescreen is a custom-made information page made with HTML, CSS, Javascript, and JQuery. This homescreen will display the current time, date, and bus times. In addition, it will also display my events for the day using the Google Calendar API. It will also dispaly the 5-day weather forecast using the Yahoo! Weather and SimpleWeather API and uses icons to display the weather conditions. Finally, it will display the top 3 news posts of the day from the Reddit /r/news subreddit using the Reddit API.

Low Poly Watchface for Pebble Time

Low Poly is a watchface made for the Pebble Time and Pebble Time Round watchface made using C and Javascript. It was designed based on the minimalism and simplicity of low poly art and has blue, red, and green backgrounds to choose from. Currently the face supports a toggleable battery meter, vibrate on disconnect, and toggleable leading zero. Feature to be added include: analog time, weather integration, Pebble Health integration, and date configuration. Download for free from the Pebble Appstore. This watchface was also featured on XNDR Tech.

Trend Dungeon

Made at the Fall 2015 UGA Hackathon, Trend Dungeon is a Twitter based text RPG that uses the Twitter API to pull current global trends and randomly generate encounters. This project was made to parody the obsession with social media trends by turning them into an interactive web game. The game works by users replying to the account's tweets and choosing what actions they would like to do. Trend Dungeon was made with Python using Tweepy. Trend Dungeon was also made in partnership with Patrick Carton and Alex Morrison.
